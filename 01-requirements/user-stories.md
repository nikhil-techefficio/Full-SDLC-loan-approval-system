# User Stories

## Applicant
US-01: As an applicant, I want to apply online
US-02: As an applicant, I want to track my application status
US-03: As an applicant, I want to upload KYC documents securely

## Loan Officer
US-03: As a loan officer, I want to review flagged applications  
US-04: As a loan officer, I want to override automated decisions with justification  

## Admin
US-05: As an admin, I want to configure business rules without code changes


## Compliance Officer
US-06: As a compliance officer, I want to audit loan decisions
US-07: As a compliance officer, I want rule-level explainability
